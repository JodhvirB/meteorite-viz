# ☄️ Meteorite Impacts

An interactive D3.js visualization exploring 32,000+ recorded meteorite landings on an interactive 3D globe.

**Live Demo:** [https://jodhvirb.github.io/meteorite-viz/](https://jodhvirb.github.io/meteorite-viz/)

## Features

- Interactive 3D orthographic globe with drag rotation and zoom
- Animated timeline playback showing meteorite discoveries over centuries
- Filter by classification (Stony, Iron, Stony-Iron), mass range, and observation type
- Canvas-based rendering for smooth 60 FPS performance
- Tooltips with detailed meteorite info on hover
- Impact burst animations during timeline playback

## Data

NASA Meteorite Landings dataset via The Meteoritical Society.

## Built With

D3.js v7, TopoJSON, HTML5 Canvas
